# MongoDB CRUD API

A simple REST API using **Node.js**, **Express**, and **MongoDB (Mongoose)** for basic CRUD operations.

---

## ✅ Features
- Create, Read, Update, Delete Items
- Uses MongoDB as database
- Tested with Postman
